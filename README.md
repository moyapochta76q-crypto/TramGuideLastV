# TramGuide

TramGuide — статический сайт-энциклопедия о трамвайных системах мира и подвижном составе.  
Сайт рассчитан на размещение на GitHub Pages и использует только HTML, CSS и ванильный JavaScript.

## Структура проекта

```text
.
├── index.html              # Главная страница
├── systems.html            # Каталог трамвайных систем
├── system-detail.html      # Страница отдельной трамвайной системы
├── vehicles.html           # Каталог вагонов
├── vehicle-detail.html     # Страница отдельного вагона
├── about.html              # О проекте
├── css
│   └── style.css           # Общие стили
├── js
│   ├── main.js             # Основная логика (навигация, страницы, данные)
│   ├── map.js              # Логика интерактивной карты на главной
│   └── filters.js          # Логика фильтрации списков
├── data
│   ├── systems.json        # Пример данных по трамвайным системам
│   └── vehicles.json       # Пример данных по подвижному составу
└── assets
    └── world-map.svg       # Упрощённая интерактивная карта мира
